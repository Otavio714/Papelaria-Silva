## 🎯 O que é?

A **Papelaria Silva** é um sistema completo de gerenciamento para papelarias que permite:

- 📦 **Gerenciar produtos** com imagens, preços e categorias
- 🚚 **Controlar fretes** por bairros/zonas
- 🛒 **Catálogo online** para clientes
- 📊 **Dashboard administrativo** com estatísticas
- 📱 **Interface responsiva** para mobile e desktop

---

## ✨ Funcionalidades

### 🛍️ Para Clientes
- **Catálogo de Produtos** - Visualize todos os produtos com imagens e preços
- **Busca e Filtros** - Encontre produtos por categoria ou nome
- **Informações de Frete** - Veja preços de entrega por bairro
- **Design Responsivo** - Funciona perfeitamente no celular e computador

### 👨‍💼 Para Administradores
- **Dashboard Completo** - Estatísticas em tempo real
- **Gestão de Produtos** - Adicionar, editar e remover produtos
- **Gestão de Categorias** - Organize produtos por categorias
- **Controle de Fretes** - Defina preços por bairro/zona
- **Interface Intuitiva** - Fácil de usar, mesmo para iniciantes

## 🚀 Como usar

### 📱 Para Clientes (Super Fácil!)

1. **Acesse o site** - Abra o arquivo `index.html` no navegador
2. **Navegue pelo catálogo** - Veja todos os produtos disponíveis
3. **Use os filtros** - Clique nas categorias para filtrar produtos
4. **Veja os fretes** - Clique em "Fretes" para ver preços de entrega
5. **Entre em contato** - Use o botão WhatsApp para fazer pedidos

### 👨‍💼 Para Administradores

#### 🔑 Primeiro Acesso
1. **Abra o site** - Arquivo `index.html`
2. **Acesse o painel**
3. **Digite a senha** 
4. **Pronto!** - Você está no painel administrativo

#### 📦 Gerenciando Produtos
1. **Clique em "Produtos"** - No painel administrativo
2. **Adicionar produto** - Clique no botão "Adicionar Produto"
3. **Preencha os dados**:
   - 📝 **Nome** - Nome do produto
   - 💰 **Preço** - Preço em reais
   - 🏷️ **Categoria** - Ex: cadernos, canetas
   - 📄 **Descrição** - Detalhes do produto
   - 🖼️ **Imagem** - URL ou arquivo do computador
4. **Salve** - Clique em "Adicionar"
5. **Pronto!** - Produto aparece no catálogo

#### 🏷️ Gerenciando Categorias
1. **Na aba "Produtos"** - Clique em "Categorias"
2. **Adicionar categoria** - Clique em "Adicionar Categoria"
3. **Preencha**:
   - 📝 **Nome** - Nome da categoria
   - 📄 **Descrição** - Descrição (opcional)
4. **Salve** - Clique em "Adicionar"

#### 🚚 Gerenciando Fretes
1. **Clique em "Fretes"** - No painel administrativo
2. **Adicionar frete** - Clique em "Adicionar Frete"
3. **Preencha**:
   - 🏘️ **Bairro** - Nome do bairro/zona
   - 💰 **Preço** - Valor do frete
4. **Salve** - Clique em "Adicionar"

## ❓ FAQ

### 🤔 Como adicionar mais produtos?
**R:** Acesse o painel administrativo → Produtos → Adicionar Produto

### 🤔 Como personalizar as cores?
**R:** Edite as classes do Tailwind CSS no arquivo `index.html`

### 🤔 Como fazer backup dos dados?
**R:** Se usar Firebase, os dados são salvos automaticamente na nuvem

### 🤔 Funciona no celular?
**R:** Sim! O site é totalmente responsivo e funciona perfeitamente no mobile

### 🤔 Como adicionar mais categorias?
**R:** Painel administrativo → Produtos → Categorias → Adicionar Categoria
